## Grocery bill (HTML/CSS/JS) → PDF

### How to run
- Open `web/index.html` in Chrome/Edge.

### Create PDF
- Click **Download PDF** → browser print dialog opens → choose **Save as PDF**.

### Notes
- This is **HTML/CSS/JS only** (no frameworks, no external PDF libraries).
- Your data is saved locally in the browser (localStorage) when you click **Save** / **Download PDF**.
